The problem

Attention cost grows with the square of context.

Self-attention computes an n × n score matrix, so memory and compute explode non-linearly as context grows. Frontier models push the limit with engineering workarounds — not by changing the underlying math.

The technology

A mathematically valid drop-in replacement.

The attention layer's input and output shapes are identical — [n, d]. Any kernel that maps Q, K, V to the correct output without materializing the n × n matrix can replace standard attention with no model retraining.

  • Near-identical accuracy. Every kernel must pass a hard MSE gate against ground-truth O(n²) output.
  • Hardware-real speed. Kernels are written in Triton / CUDA and measured on standardized GPUs — not just in theory.
  • Works with your stack. Llama-3, Mistral, Qwen, or any Transformer — swap one layer, keep your weights.
the operation that explodes
Attention(Q, K, V) = softmax( Q · KT / √d ) · V

The bottleneck is a single intermediate step — Q · KT, the n × n score matrix. Because it is only intermediate, replacing it is provably possible.

Deterministic evaluation

Scored by physics, not opinion.

Every submission passes through a fully automated, three-tier gate. No language model ever judges another — ranking is pure measurement, reproducible by anyone.

Tier 1 · Accuracy gate

Mean Squared Error against ground-truth attention. Exceed the precision budget and the kernel scores zero — speed is irrelevant if the output is wrong.

Tier 2 · Scaling proof

Re-run at 128K–256K tokens on locked hardware. Double the sequence length: if latency quadruples, it isn't sub-quadratic — and it's rejected.

Tier 3 · Composite score

Among kernels that pass both gates, the highest accuracy × 1/VRAM × 1/latency takes 100% of the prize. Winner-take-all.
